## 技术栈

nodejs + express + mysql


## 项目运行

```
项目运行之前，请确保系统已经安装以下应用
1、node (16.0 及以上版本)
2、mysql (开启状态)
```

```
git clone https://gitee.com/ZenDevon/client_pc_server.git  

cd client_pc_server

npm install

npm run start

```
